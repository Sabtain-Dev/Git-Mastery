# Day 17 Notes
This file is related to GitHub Issues practice.

1️⃣ Core Concept — What is an Issue?

An Issue is:

A task, bug report, or feature request inside a repository.

It is NOT code — it’s discussion + tracking.

2️⃣ Why Issues Are Important

In real projects, Issues are used to:

Report bugs 🐞
Request features 🚀
Assign tasks 👨‍💻
Track progress 📊

3️⃣ Types of Issues
Type	  Example
Bug	      “Login button not working”
Feature	  “Add dark mode”
Task	  “Update README”
